E-Waste Management and Recycling System
(A Web-Based Platform for Responsible Electronic Waste Disposal)

Frontend Technologies:
  1. HTML5: Semantic structure and accessibility features
  2. CSS3: Responsive design with Bootstrap 5.1.3 framework
  3. JavaScript: Client-side validation and interactive features
  4. Bootstrap Components:
    o Responsive grid system
    o Modal dialogs for detailed views
    o Form validation and styling
    o Navigation components

Backend Technologies:
  1. PHP 7.4+: Server-side logic and business rules implementation
  2. MySQL 5.7+: Relational database management
  3. Apache HTTP Server: Web server environment

Development Tools:
  1. Visual Studio Code: Primary code editor with PHP extensions
  2. XAMPP Stack: Local development environment (Apache, PHP, MySQL)
  3. phpMyAdmin: Database management interface
